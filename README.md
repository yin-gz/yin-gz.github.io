# Gongzhu Yin Academic Homepage

Static academic homepage ready for GitHub Pages.

## Files

- `index.html`: page structure and profile sections.
- `styles.css`: responsive academic layout.
- `script.js`: publication data, rendering, and filters.
- `assets/`: portraits and publication visual summaries.

## Publish

Push these files to a GitHub Pages repository and enable Pages from the repository settings. For a personal page, use a repository named `yin-gz.github.io`.

## Update Publications

Edit the `publications` array in `script.js`. Replace files in `assets/pubs/` with real paper figures using the same filenames if desired.
